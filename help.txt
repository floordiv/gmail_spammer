используется ЯП Python, модуль smtplib

в папке, в которой будет лежать сам скрипт spammer.py и shell.py, также должны быть три файла:

    > proxies.txt - в нём построчно будут записаны https (важно) прокси

    > mails.txt - в нём будут лежать почтовые ящики, с которых должен будет производиться спам (в формате адрес/пароль)

    > targets.txt - файл, в котором будут лежать сами цели спама - т.е. электронные адреса

    > (необязательно) help.txt - документация к программе

spammer.py - небольшая программа, которая будет производить спам

shell.py - командная строка для управления программой

при первом запуске, в файле proxies.txt уже будут лежать прокси

для запуска, в зависимости от операционной системы и настроек, введите:

   windows/cmd (или любой другой терминал)/python3 : python shell.py  (находясь при этом в папке с самой программой; при возникновении ошибок, попробуйте вместо python ввести python3)

   linux/python3 : python3 shell.py (всё также находясь в одной папке с программой)
